# [SMWU 2020-1] 도서관 홈페이지 제작하기 
##### #데이터베이스프로그래밍 프로젝트 #대학교도서관서비스
<br><br>
JSP와 JDBC,PL/SQL을 사용해 **데이터베이스와 웹이 연동된 프로그램**을 만들어보기 위해 도서관 서비스를 구현해보았다.<br>

대학교 도서관 홈페이지에서는 다양한 서비스를 제공하는데 우리는 그 중 **도서 서비스**와 **좌석 서비스**를 구현했다.
